При помощи данного API можно:
  • Получить полный список всех внесенных репортажей о погоде: get запрос по адресу - /api/weather
  • Получить данные о репортаже по его уникальному индификатору: get запрос по адресу - /api/weather/<int:weather_id>
  • Удалить репортаж по его уникальному индификатор: delete запрос по адресу - /api/weather/<int:weather_id>
  • Создать новый репортаж: post запрос по адресу - /api/weather
При создании нового репортажа нужно придерживаться следующей структуры файла json:

{
  "location_id": 1,
  "date": "2023-04-24",
  "clouds": 1,
  "temperature": 12,
  "water_temperature": 11,
  "precipitation_type": "rain",
  "precipitation_value": 900,
  "wind_direction": "E",
  "wind_velocity": 90,
  "atmospheric_pressure": 765
}
